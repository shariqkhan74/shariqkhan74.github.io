# 🌐 Mohd Shariq Ali Khan – Portfolio

A modern, responsive **developer portfolio website** built using **HTML, CSS, and JavaScript**.  
It showcases my skills, projects, education, and professional experience as a **Full-Stack Web Developer**.

---

## ✨ Overview

This portfolio is designed with a **glassmorphism gradient theme**, **smooth scroll animations**, and a **mobile-first layout**.  
It presents a clean, elegant interface suitable for developers and professionals looking to highlight their work.

---

## 🔧 Features

- 🧭 **Hero Section** – Name, title, tagline & call-to-action  
- 👨‍💻 **About Me** – Summary based on professional background  
- ⚙️ **Skills** – Displayed as skill tags & progress bars  
- 💼 **Projects** – Key works with titles, stacks, and short descriptions  
- 🧱 **Experience** – Professional roles & responsibilities  
- 🎓 **Education** – Academic qualifications & certifications  
- 📬 **Contact** – Simple mailto contact form  
- 🌈 Gradient & glassmorphism design  
- 📱 Fully responsive for desktop, tablet, and mobile  
- ⚡ Smooth scroll & fade-in animations
